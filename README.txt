

APPOINTMENT SCHEDULER

Author: Corey Hall
Version: 1.0
01-07-2022


ABOUT:
The Scheduler is a GUI-based scheduling application for managing the appointments of customers, contacts, and users in various timezones.


SPECIFICATIONS:
IDE: IntelliJ IDEA 2021.1.1 (Community Edition) - Build #IC-211.7142.45, built on April 30, 2021
Java Version: Java SE 11.0.11
JavaFX SDK: 11.0.2
JDBC Driver: mysql-connector-java-8.0.25
